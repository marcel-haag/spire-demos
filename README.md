# spire-demos
A demo collection for stateful and stateless workloads using SPIRE, SPIFFE, Envoy, OPA, and Kubernetes.


Stateful: 
- Datenbank 
- SPIRE Server mit Registration Entries / Trust Bundle / CA State 
- Frontend-Service 
- Backend-Service 
- Envoy Sidecar 
- OPA Sidecar 
- Demo-App ohne persistente Daten

Stateless: 
- Operator
- Helm CHarts
